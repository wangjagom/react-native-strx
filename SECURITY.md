# Security Policy

## Supported Versions

Only the latest published version of `react-native-strx` is supported with security fixes.

## Reporting a Vulnerability

Please report security vulnerabilities privately through GitHub Security Advisories when available:

https://github.com/wangjagom/react-native-strx/security/advisories/new

If GitHub Security Advisories are unavailable, open a minimal GitHub issue that does not include exploit details, and ask for a private disclosure channel.

Please include:

- affected version
- minimal reproduction
- expected impact
- relevant environment details

Do not publicly disclose vulnerability details until a fix is available.
